# UNSEEN-LAYERS
School project  
Plot: The player is an ordinary person desperate to regain their self (personality) within 18 hours, and must balance "time" and "energy" through daily choices such as cooking, ordering takeout, cleaning, and resting. After 18 hours, this ordinary person will die in this mental world; if the mental world collapses, the protagonist will never wake up again.  
Theme (Beneath the Surface): This so-called "apartment" is actually the mental world locked down by the main personality after suffering severe trauma. The player is not the owner, but a newly split "protective/compensatory personality." After 18 hours, the main personality will face "mental brain death."
